Reflection on the CS-370 Portfolio

For this project, I created an intelligent pirate agent that solves a treasure hunt game's pathfinding problem using a deep Q-learning algorithm. The goal was to teach the pirate NPC how to find the loot before the human player by navigating a maze with obstacles and passageways. The beginning code, which included the neural network framework, auxiliary functions, maze setup, game environment, and accompanying Python files, was supplied by the teacher. I was told not to change any of the supplied.py scripts. Rather, I finished the TreasureHuntGame.ipynb Jupyter Notebook's Q-Training Algorithm part.

The deep Q-learning process was implemented by the code I wrote, which included training the neural network from replay memory, storing experiences, choosing actions based on an exploration versus exploitation strategy, updating Q-values, and repeatedly training the pirate agent until it consistently located the treasure. In order to make the program easier to read and maintain, I also made sure it ran error-free and included comments.

Why is computer science important, and what do computer scientists do?

Computer scientists create, build, and enhance software systems that address practical issues. They develop algorithms, conduct data analysis, and construct applications that increase the effectiveness and use of technology. Their work has an impact on almost every business, including healthcare, banking, entertainment, and transportation. By enabling an agent to learn from experience rather than depending on hard-coded instructions, this study showed how artificial intelligence may be utilized to address complicated decision-making challenges. 

As a computer scientist, how do I approach a problem?

Before deciding what needs to be built, I address problems by first comprehending the requirements and identifying what has previously been supplied. After that, I divide the task into manageable chunks, apply each component separately, and regularly test the code to make sure it functions as intended. I thoroughly examined the initial code for this project before putting the Q-training approach into practice. To increase the pirate agent's capacity to locate the riches, I relied on testing, debugging, and modifying the training procedure.

What ethical obligations do I have to the organization and the end user?

It is my duty as a computer scientist to create software that is precise, dependable, safe, and maintained. Making ensuring algorithms avoid prejudice, make fair decisions, and function as intended without inflicting unintentional harm is crucial when working with artificial intelligence. It is also my duty to write clear, well-documented code that can be maintained by others and, when necessary, to safeguard user data. Organizations can create reliable systems and give end users satisfying experiences by adhering to ethical and professional standards.
